# Real-Time Object Detection

This project detects objects in real time using a webcam/video stream.

## Features
- Real-time detection using YOLOv8
- Works with webcam or images
- Fast and accurate

## Technologies
- Python
- OpenCV
- YOLOv8

## How to Run
1. Install requirements:
   pip install ultralytics opencv-python

2. Run:
   python detect.py

## Output
The system detects objects like person, car, etc. in real-time.
